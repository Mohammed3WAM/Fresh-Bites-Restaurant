# FreshBites Digital Menu

A premium, interactive digital restaurant menu designed to maximize conversion and appetite appeal.

## Features

- **Irresistible Visuals**: Dark premium theme with vibrant accents and high-quality imagery.
- **Dynamic Filtering**: Instantly filter between Breakfast, Lunch, and Drinks without reloading.
- **WhatsApp Ordering**: Build your order and send it directly to the restaurant via WhatsApp with one click.
- **Bilingual Support**: Fully functional Arabic/English toggle with RTL support.
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop.

## How to Use

1. **Open the Menu**: Simply open `index.html` in any modern web browser.
2. **Customize Menu**: Edit the `menuItems` array in `script.js` to add your own dishes and prices.
3. **Setup WhatsApp**: In `script.js`, change the `phone` variable to your restaurant's WhatsApp number.
4. **Photos**: Replace images in the `images/` folder or update the paths in `script.js`.

## Tech Stack
- HTML5
- CSS3 (Variables, Flexbox, Grid, Animations)
- Vanilla JavaScript
- FontAwesome (Icons)
- Google Fonts (Outfit)
